# grapisocialnetwork
A Facebook-like social network.

## Dependencies
NodeJS and MongoDB

## Web app
Open:
* 1_installWebsite
* 3_startMongoDB(OSX or Windows)
* 4_runWebsiteServer
* [http://localhost:3000](http://localhost:3000)

## Mobile app
Open:
* 1_installMobileApp
* 2_installIonic(OSX or Windows)
* 3_startMongoDB(OSX or Windows)
* 4_runMobileAppServer
* 5_previewMobileApp
